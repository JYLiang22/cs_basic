# 文件系统基础知识


## 文件系统的基本组成

首先介绍了文件系统使用索引节点(inode)和目录项(dentry)组成的，搞清楚这两个元素里面包含了什么东西、对应关系、分别存放在哪里(磁盘or内存)。<br>
注意区分目录和目录项，和局部性原理密切相关。<br>
磁盘在进行格式化是会分为哪几个存储区域、这几个存储区域分别存放了什么、这几个区域何时会被加入内存。<br>

## 文件的使用

读取文件的过程：open -> write -> close。<br>
为什么要提出打开文件表，它和进程的对应关系是什么，它里面每一项都是什么东西，操作系统在里面维护了文件的哪些状态和信息。<br>

## 文件的存储

分为连续空间存放方式和非连续空间存放方式，其中非连续又分为链表方式和索引方式，其中链表又分为隐式链表和显式链表。<br>
搞清楚以上各种方式的存放原理(即使用什么数据结构存放)、该存放原理导致的优缺点、各自特点。<br>
