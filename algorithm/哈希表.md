# 哈希表基础理论
关于什么是哈希表、哈希表工作原理、哈希函数等的就不多赘述了，数据结构课上已经学过了。


## 常见的三种哈希结构
这里主要有三种数据结构：数组、set(集合)、map(映射)。数组就不多赘述了，set和map主要看cpp primer。<br>

这里主要介绍使用场景：
1. 如果哈希表的大小可以确定，可以使用数组，因为数组操作简单
2. 优先使用unordered_set，因为它的查询和增删效率是最优的
3. 如果需要集合是有序的，那么就用set
4. 如果要求不仅有序还要有重复数据的话，那么就用multiset


# 做题步骤

如果确定了题目要用哈希方法，可以按以下步骤思考：
1. 使用哪一种数据结构？数组？set？map？
2. 目前做过的题中，使用数组的情况都是思路比较简单的
3. 使用set，是使用unordered_set、multiset还是set？set用来存什么？set中的key用来存什么？
4. 使用map，是使用unordered_map、multimap还是map？map用来存什么？map中的key和value用来存什么？
