# 问题引入
通过面试过程中面试官水平不够，引出了这个问题，并直接给出了结论：一般情况下，不管 TLS 握手次数如何，都得先经过 TCP 三次握手后才能进行，因为 HTTPS 都是基于 TCP 传输协议实现的，得先建立完可靠的 TCP 连接才能做 TLS 握手的事情。<br>

但是面试官说的场景也是可能发生的，不过要满足一定条件：
1. 客户端和服务端都开启了 TCP Fast Open 功能，且 TLS 版本是 1.3
2. 客户端和服务端已经完成过一次通信
